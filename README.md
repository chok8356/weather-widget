# weather-widget

## Demo
![](demo.gif)

## Project setup
```
npm install
```

Insert API key from [openweathermap.org](https://openweathermap.org/) in the `.env` file

```
VUE_APP_API_KEY=YOUR_API_KEY
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build:wc
```
